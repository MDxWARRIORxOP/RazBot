RazBot: A Discord bot written in discord.py (python):


Here you can download the bot and see updates about it. Note: RazBot is mainly meant for use on my server KingdomsCrusade, but I decided to make it public on here to help other people learn how to make a very basic bot. To work on other servers, some stuff will need to be changed like some emoji ids for "raz!vote" and the "raz!lucky" ping id would need to change to whoever is handling the giveaway. I might make and host a public version of the bot meant to work on all Discord servers if enough people want it.

How to download:

Download the .zip below, extract it, go to main.py and at the bottom of the file you should see "client.run("TOKEN")". Where it says TOKEN place your bot token from the Discord Developer Portal.

How to run it:

Make sure you have installed (pip install ...) "discord" and "asyncio". Then run main.py (to run on a hosting using a panel, which is probably linux, type "python main.py" to start it).
